Google User Data Deletion Notice

Users may request permanent deletion of personal information and Google user data associated with Wheedle Technologies services.

Deletion requests are generally processed within 30 business days.

Wheedle Technologies does not sell Google user data and does not use Google user data for advertising, AI training, or unrelated commercial purposes.

How to Request Deletion

To request deletion of your information, email:

info@wheedletechnologies.ai

Use the subject line:

Delete My Data

Information to Include

  • Full name
  • Email address
  • Google account email (if applicable)
  • Description of deletion request

Verification Process

We may verify user identity before processing deletion requests to protect accounts and prevent unauthorized requests.

Data That May Be Retained

Certain information may be retained where required for:

  • Legal compliance
  • Fraud prevention
  • Security obligations
  • Dispute resolution

Google User Data Handling

Google user data accessed through Google APIs is used only for providing requested functionality.

We do not:

  • Sell Google user data
  • Use Google user data for advertising
  • Use Google user data for AI training
  • Transfer Google user data to data brokers
